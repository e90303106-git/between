# 留白之間 between｜第二版品牌形象網站

本版依照指定參考畫面重新製作，並保留第一版的網站主分類，不新增其他分類。

## 主分類
- 留白自己
- 留白關係
- 留白工作
- 關於我

## 視覺調整
- 左上識別改為純留白圓形，不放文字、不保留紅框。
- 首頁使用個人形象插畫，衣服改為蘋果綠，搭配奶油黃主視覺。
- 整體風格維持專業、溫暖、簡潔與留白感。

## 使用方式
1. 將整個 `between` 資料夾複製到 Windows 電腦的 `D:\between`。
2. 雙擊 `D:\between\index.html` 即可瀏覽。

## 專案內容
- `index.html`：網站內容與結構
- `styles.css`：網站版面與視覺樣式
- `script.js`：手機選單、滾動動畫與導覽狀態
- `assets/avatar-green.png`：網站使用的個人形象插畫
- `assets/avatar-original.jpg`：原始形象圖備份
- `assets/reference-homepage.png`：版面參考圖
